1. Torrent Box

Download torrents on the Raspberry Pi. Completed downloads can be found in the /torrents/complete/ folder of the network drive.

Torrent Control Interface

Transmission Bittorrent Web Interface

Chrome Extension

Easily add magnet links to the Raspberry Pi Torrent Box using this extension for Google Chrome.

After installation right click on it's icon and select 'Options'. Set the following options:

Option Set To:
RPC address http://192.168.1.2:9091/transmission/rpc
WebUI address http://192.168.1.2:9091

To add a torrent simply right click on the magnet link and select '.torrent to Transmission'.

2. Network Drive

The Raspberry Pi has network storage. To use this in Windows go to 'My Computer','Network','Raspberry Pi','Public'. Right clicking on 'Public' will give you the option of mapping the network storage to a drive letter(for example, Z:\).

3. Media Server

To use the Media Server use a DLNA compliant client. Searching for DLNA on Gogle Play and the Apple App Store will give you good results. Some clients, such as Allshare for Android are able to stream the media to the Chromecast.
